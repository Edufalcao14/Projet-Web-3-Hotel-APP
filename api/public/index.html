Welcome to Vinci Hotel API

API Documentation

Documentation is available at http://localhost:8080/api-docs/

Available Endpoints

- Rooms

  • GET /rooms - Retrieve all rooms
  • GET /rooms/stats - Retrieve today's room statistics
  • GET /rooms/stats/:date - Retrieve room statistics for a specific date

- Finance

  • GET /finance/prediction/:date - Retrieve finance prediction for a specific date's month
  • GET /finance/revenue/:days - Retrieve finance revenue for the last n days
  • GET /finance/revenue/stats/:days - Retrieve finance revenue statistics for the last n days
  • GET /finance/revenue/stats/year/:year - Retrieve finance revenue statistics for a specific year
