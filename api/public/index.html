API Documentation

Documentation is available at http://localhost:8080/api-docs/

Available Endpoints

- Rooms

  • GET /rooms - Retrieve all rooms

- Statistics

  • GET /stats/rooms - Retrieve today's rooms statistics
  • GET /stats/rooms/yyyy-mm-dd - Retrieve rooms statistics for a specific date
